# Daily Flow 开发计划审批摘要

日期：2026-06-12  
状态：已批准执行阶段 0

## 产品

开发一款完全开源的安卓日程与生活记录应用，统一提供：

- 任务、日历事件、完成状态、重复规则和多重提醒。
- 可自定义的多选、计数、滑条、布尔、时长、数值和文字记录。
- 日、周、月统计、标签分布、连续天数和 AI 总结。
- 本地输入法语音输入。
- DeepSeek 自然语言意图识别。
- AI 写入前确认和日期歧义追问。
- 本地优先、无广告、无追踪、API Key 安全存储。

## 开源方案

- 主干 Fork：[My Brain](https://github.com/mhss1/MyBrain)，复用任务、日历、提醒、日记、AI Agent、Room、Compose 和 Koin。
- 选择性移植：[Track & Graph](https://github.com/SamAmco/track-and-graph)，复用时间序列、快捷记录、聚合、图表及测试思路。
- 两者均为 GPLv3 兼容许可证，Daily Flow 继续采用 GPLv3。
- 不整体合并 Track & Graph，避免双数据库、双依赖注入和维护成本。

## 首版范围

`v1.0` 包含：

- Android 8 及以上。
- 单一应用。
- 结构化事件模板和八种字段类型。
- 任务与日历事件的多个提醒。
- DeepSeek 连接、意图识别、确认卡和工具执行。
- 趋势统计、CSV 导出、普通备份恢复。
- Android Keystore、生物识别锁和 HTTPS 限制。
- GitHub APK，并提交 F-Droid。

`v1.0` 不包含：

- CalDAV。
- 数据库加密和加密备份。
- 内置语音识别。
- 云同步、多人协作、iOS 和网页端。

数据库加密与加密备份计划进入 `v1.1`，CalDAV 计划进入 `v1.2`。

## 阶段

1. Fork、CI、许可证和 DeepSeek 技术验证。
2. 结构化记录数据模型。
3. 模板、动态表单、快捷录入和历史。
4. 统计图表和 AI 总结。
5. 统一多提醒。
6. DeepSeek 提案、确认卡和工具。
7. Keystore、应用锁和备份恢复。
8. 测试、性能、文档、GitHub 和 F-Droid 发布。

## 关键控制

- AI 只能生成操作提案，用户确认后才写入。
- 日期和提醒不明确时必须追问。
- API Key 使用 Android Keystore，不进入日志或备份。
- 核心功能无网络可用，DeepSeek 故障不影响日程、提醒和记录。
- 模板字段采用停用和快照机制，避免破坏历史数据。
- 移植代码保留版权头，并建立第三方代码来源台账。
