## 1. 欢迎使用 Daily Flow

**Daily Flow** 是一款完全开源、本地优先的 Android 个人效率应用。它将以下功能整合在一个应用中：

- ✅ **任务管理** —— 创建、编辑、完成待办事项，支持优先级、子任务和重复规则
- ✅ **日历事件** —— 与系统日历同步，月视图浏览
- ✅ **结构化记录** —— 自定义模板，支持 8 种字段类型（多选、单选、计数、滑条、布尔、时长、数值、文本）
- ✅ **笔记 / 日记 / 书签** —— 统一内容库入口
- ✅ **AI 助手** —— 通过 DeepSeek 等 AI 模型用自然语言创建任务、事件和记录（写入前需确认）
- ✅ **多重提醒** —— 任务和日历事件支持多个提前提醒
- ✅ **统计图表** —— 折线图、柱状图、饼图，日/周/月聚合
- ✅ **全局搜索** —— 跨类型搜索所有内容
- ✅ **桌面小组件** —— 快捷记录常用模板
- ✅ **本地优先** —— 无网络时所有核心功能正常使用
- ✅ **安全存储** —— API Key 使用 Android Keystore 加密，不上传、不追踪、无广告

## 2. 安装与初次启动

### 安装方式

Daily Flow 提供以下安装渠道：

- **GitHub Release APK**：从 [GitHub Releases](https://github.com) 下载 `.apk` 文件，在 Android 设备上直接安装
- **F-Droid**（即将上线）：通过 F-Droid 开源应用商店安装

### 系统要求

| 项目 | 要求 |
|------|------|
| Android 版本 | 8.0 (API 26) 及以上 |
| 存储空间 | 约 10 MB（应用本体） + 数据存储 |
| 网络 | 核心功能离线可用；AI 助手需要网络连接 |
| 权限 | 日历权限（可选，用于系统日历集成）；通知权限（可选，用于提醒） |

### 初次启动

首次打开 Daily Flow，您会看到**空间**页面，显示 2×2 功能卡片网格。底部导航栏有四个标签：**概览、日历、空间、设置**。

> 💡 **提示**：如果您之前使用过 My Brain 应用，Daily Flow 是基于 My Brain v3.1.0 Fork 开发的继任者，继承了其核心架构并新增了大量功能。

## 4. 概览 —— 今日一览

点击底部 **概览** 标签，可查看当天的综合信息：

### 页面内容（从上到下）

1. **顶栏** —— 标题"概览" + 右侧搜索图标
2. **自定义记录卡片** —— 显示常用模板（健身、心情、习惯等）的快捷记录按钮，以及"今天 N 条记录"的计数
3. **日历卡片** —— 显示今日系统日历事件摘要。如未授权日历权限，会显示"获取权限"提示
4. **待办事项卡片** —— 今日任务列表 + 添加按钮
5. **情绪统计 + 事项摘要** —— 日记心情分布和近期事项统计
6. **Pending Reminders 卡片** —— 即将到来的提醒列表，按目标类型（任务/日历事件/记录提示）分组显示

### 快捷操作

- 点击搜索图标 → 进入**全局搜索**
- 点击自定义记录中的模板按钮 → 直接进入**快速记录**页面

## 5. 日历 —— 月视图与事件管理

点击底部 **日历** 标签进入完整月历视图。

### 月视图功能

- **月份切换**：左右滑动或点击箭头切换月份
- **日期格**：每天显示
  - 🔵 蓝色圆点：系统日历事件
  - 🟠 橙色指示条：本地自定义记录
- **点击日期**：展开当日详情列表
- **列表/月历切换**：可在月视图和列表视图之间切换
- **日历账户筛选**：如设备有多个日历账户，可筛选显示

### 新增事件

点击日历页右上角 **+** 按钮，弹出类型选择：

- **新增日历事件** → 写入 Android 系统 Calendar Provider，可在其他日历应用看到
- **添加自定义记录** → 写入 Daily Flow 本地 tracking 数据库

在日历事件编辑页面，您可以：

- 设置标题、开始/结束时间
- 添加**多个提醒**（准时、提前 5/15/30 分钟、1 小时、1 天，或自定义绝对时间）
- 选择日历账户

> ⚠️ **权限说明**：日历功能需要设备日历读写权限。如未授权，日历卡片会显示权限请求提示，不影响其他功能。

## 6. 空间 —— 全部功能入口

点击底部 **空间** 标签，显示 2×2 功能卡片网格：

| 位置 | 卡片 | 功能 |
|------|------|------|
| 左上 | **内容库** | 笔记、日记、书签的统一入口 |
| 右上 | **自定义记录** | 模板管理、快速记录、历史、统计 |
| 左下 | **待办事项** | 任务列表、创建、编辑、多提醒 |
| 右下 | **Assistant** | AI 助手，自然语言交互 |

### 6.1 待办事项（任务）

#### 任务列表

- 显示所有待办任务，支持排序
- 每个任务显示：标题、截止日期、优先级、提醒标记
- 点击任务进入详情

#### 创建任务

1. 点击 **+** 按钮
2. 输入任务标题
3. 可选设置：
   - **截止日期**：选择日期和时间
   - **优先级**：高/中/低
   - **重复规则**：每天/每周/每月/每年/自定义
   - **子任务**：添加检查清单
   - **多提醒**：点击"添加提醒"，选择快捷选项或自定义时间
4. 点击保存

#### 多提醒功能

每个任务和日历事件支持**多个提醒**：

- **快捷选项**：准时、提前 5/15/30 分钟、1 小时、1 天
- **自定义绝对时间**：精确到分钟的任意时间
- **相对提醒**：当修改截止时间后，相对提醒自动重算
- **防重复**：系统阻止添加相同时间的重复提醒
- 完成任务后，所有提醒自动取消

### 6.2 内容库（笔记/日记/书签）

内容库将笔记、日记和书签统一为一个入口。

#### 筛选与浏览

- **顶部筛选栏**：全部 / 笔记 / 日记 / 书签，一键切换
- **搜索框**：搜索所有内容
- **排序**：按创建时间、更新时间、置顶状态

#### 创建内容

点击 **+** 按钮，选择内容类型：

- **笔记**：标题 + 正文，支持文件夹和外部 Markdown
- **日记**：标题 + 正文 + 心情（五档）+ 发生日期
- **书签（链接）**：标题 + URL + 描述

> 💡 **提示**：创建时默认是笔记。添加心情后自动保存为日记；添加 URL 后自动保存为书签。

#### 编辑与删除

- 点击内容条目进入详情/编辑
- 删除时显示确认对话框
- 删除后出现 **撤销 Snackbar**，可在几秒内恢复

## 7. 自定义记录详解

自定义记录是 Daily Flow 的核心创新功能。您可以创建**模板**来定义要记录的内容，然后用模板快速录入。

### 支持的 8 种字段类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **多选 (Multi-Select)** | 同时选择多个选项 | 健身部位：胸、背、臀、腿 |
| **单选 (Single-Select)** | 选择一个选项 | 心情：好、一般、差 |
| **计数 (Counter)** | 点击增加/减少 | 吸烟次数、喝水杯数 |
| **滑条 (Scale)** | 滑动选择数值 | 心情评分 1-10 |
| **布尔 (Boolean)** | 是/否开关 | 是否服药 |
| **时长 (Duration)** | 小时:分钟 | 运动 60 分钟 |
| **数值 (Number)** | 数字输入 | 体重 70.5 kg、饮水量 2000 ml |
| **文本 (Text)** | 自由文本 | 备注、感想 |

### 7.1 模板管理

进入：**空间 → 自定义记录**，显示模板列表。

#### 内置模板

Daily Flow 预置三个模板，开箱即用：

- 🏃 **健身**：部位（多选）、时长、备注
- 😊 **心情**：心情等级（滑条 1-10）、备注
- 🔢 **习惯次数**：习惯名称（单选）、次数（计数）

#### 模板操作

- **创建模板**：点击右上角 **+** → 设置名称、图标、颜色 → 添加字段 → 保存
- **复制模板**：长按模板 → 复制，快速创建变体
- **编辑模板**：点击模板 → 编辑，修改字段、选项
- **排序**：拖拽调整模板显示顺序
- **停用**：不再使用的模板可停用（数据保留，仅隐藏）
- **固定**：常用模板可固定在列表顶部

#### 模板编辑器

在编辑器中可以：

- 修改模板名称、图标、颜色
- **添加字段**：选择 8 种类型之一 → 设置字段名、单位、必填等
- **编辑选项**（多选/单选字段）：添加、排序、停用选项
- **预览**：查看模板在实际记录中的效果

> ⚠️ **重要**：已使用的模板/字段/选项只**停用**不删除，避免破坏历史数据。

### 7.2 快速记录

进入：**空间 → 自定义记录 → 点击模板**，或从**概览页快捷按钮**进入。

#### 记录页面

- 显示模板名称和所有字段
- 每个字段渲染为对应的输入控件
- **发生时间**：默认为当前时间，可修改（支持补录）
- **备注**：可选的文字备注
- 保存按钮

#### 智能建议

- **默认值**：字段可设置默认值，快速录入
- **最近使用**：选项按最近使用频率排序
- **计数递增**：计数器显示上次值，支持一键 +1
- **文本历史**：文本字段显示历史输入，快速选择

#### 保存与继续

保存成功后：

- 显示成功提示（含 session ID）
- 可选择"继续记录"（相同模板再次录入）
- 或返回模板列表

### 7.3 记录历史

进入：**空间 → 自定义记录 → 点击模板 → 历史**。

#### 查看历史

- 按时间倒序显示所有记录
- 每条显示：发生时间 + 各字段的值
- 可**按模板/日期范围筛选**

#### 操作历史

- **点击**：查看详情
- **编辑**：修改记录值和发生时间
- **删除**：确认后删除（显示撤销 Snackbar）

### 7.4 统计与图表

进入：**空间 → 自定义记录 → 统计**，或从模板列表进入。

#### 统计视图

- **范围切换**：日 / 周 / 月 / 年
- **Tracker 选择器**：选择要分析的字段
- **聚合方式**：总次数 / 总量 / 平均值 / 最大值 / 最小值

#### 图表类型

| 图表 | 说明 |
|------|------|
| 📈 **折线图** | 趋势变化，适合数值和滑条字段 |
| 📊 **柱状图** | 分类对比，适合计数和时长 |
| 🥧 **饼图** | 选项分布，适合多选和单选字段 |
| 📋 **分布表** | 数值分布详情 |

#### 统计指标

- **连续天数**：当前连续记录天数 + 最长连续天数
- **今日摘要**：今日各模板记录次数汇总
- **空 bin 填充**：没有记录的日期也显示为 0，保持图表连续
- **DST 处理**：正确处理夏令时切换

### 7.5 CSV 导入导出

进入：模板列表页 → CSV 入口。

#### 导出

- 将所有记录导出为单个 CSV 文件（UTF-8 BOM，ISO 8601 日期格式）
- 文件保存到设备下载目录
- 可在 Excel / Google Sheets 中直接打开

#### 导入

1. 选择 CSV 文件
2. 预览：显示将导入的记录数量、模板分布
3. 确认后原子导入（全部成功或全部回滚）
4. 支持增量导入，不会重复已有记录

### 使用方式

2. 选择"添加小组件"

## 10. 设置

点击底部 **设置** 标签进入。

### 10.1 外观与主题

- **主题切换**：亮色 / 暗色 / 跟随系统
- **字体选择**：多种字体可选
- **Material You**（Android 12+）：动态取色，跟随壁纸
- **语言**：跟随系统

### 10.2 AI 集成与 Provider 配置

Daily Flow 支持连接多个 AI 模型 Provider，通过 OpenAI 兼容协议。

#### Provider 列表（分组显示）

| 分组 | Provider | 默认模型 |
|------|----------|----------|
| ⭐ **推荐** | DeepSeek | deepseek-v4-flash |
| ⭐ **推荐** | OpenAI | gpt-5.5 |
| 🇨🇳 **中国大陆** | 通义千问 | qwen-plus |
| 🇨🇳 **中国大陆** | Kimi (月之暗面) | kimi-k2.6 |
| 🇨🇳 **中国大陆** | 智谱 GLM | glm-5.1 |
| 🌍 **国际** | Gemini | — |
| 🌍 **国际** | Anthropic | — |
| 🌍 **国际** | OpenRouter | — |
| 💻 **本地** | Ollama | — |
| 💻 **本地** | LM Studio | — |

#### 配置步骤

1. 在设置页找到 AI 集成入口
2. 选择 Provider → 点击展开
3. 填入 **API Key**（从 Provider 官网获取）
4. 可选修改 **模型名称** 和 **Base URL**
5. 首次启用时默认选中 DeepSeek

> 🔒 **安全**：API Key 使用 Android Keystore (AES/GCM) 加密存储，不会出现在日志、备份或 DataStore 明文中。

#### AI 助手使用

进入：**空间 → Assistant**。

- 用自然语言输入需求，例如：
  - "明天下午 3 点开会"
  - "创建一个每天跑步的提醒"
  - "记录今天健身 60 分钟"
  - "今天心情怎么样？帮我记日记"
- AI 会生成**确认卡**，显示将执行的操作
- 您可以**确认、编辑或取消**
- 所有写操作必须经过确认才执行
- 日期不明确时 AI 会追问

> 💡 **隐私保护**：AI 总结功能只发送用户选择的统计汇总数据，不上传原始记录。

### 10.3 安全：应用锁

- **开关**：设置 → 应用锁 → 开启
- **锁定方式**：设备凭据（PIN/图案/密码）或生物识别（指纹/面部）
- **后台超时**：切换到后台超时后自动锁定
- **任务缩略图**：锁定期间遮挡最近任务中的缩略图
- **提醒友好**：锁屏期间提醒通知正常触发
- 关闭开关后不再显示锁屏

### 10.4 备份与恢复

#### 导出备份

1. 设置 → 备份 → 导出 JSON
2. 备份文件保存到下载目录
3. 包含：所有任务、日历事件、自定义记录、笔记、日记、书签、提醒
4. 不包含：API Key、Keystore 密文、临时提案

#### 恢复备份

1. 设置 → 备份 → 导入
2. 选择 JSON 备份文件
3. **预览**：显示数据计数和版本信息
4. 确认后**原子恢复**（失败则全部回滚）

> ⚠️ **版本要求**：备份文件包含 `schemaVersion`，跨大版本恢复前会验证兼容性。

## 11. 安全与隐私

Daily Flow 以隐私为首要设计原则：

| 措施 | 说明 |
|------|------|
| **本地存储** | 所有数据默认仅保存在设备本地 |
| **API Key 加密** | 使用 Android Keystore (AES/GCM)，不进入日志、备份、DataStore 明文 |
| **应用锁** | 支持设备凭据和生物识别 |
| **HTTPS 强制** | 发布版只允许 HTTPS API 地址 |
| **无追踪** | 不集成广告、追踪或崩溃分析 SDK |
| **数据预览** | 即将发往 AI 的数据可预览 |
| **AI 总结安全** | 只发送用户选定的汇总数据 |
| **备份排除** | 备份不包含 API Key 和敏感密文 |
| **日志脱敏** | AI 请求日志自动脱敏 API Key |

## 12. 常见问题

### Q: 为什么日历看不到事件？

A: 需要授予日历读写权限。前往设置 → 应用 → Daily Flow → 权限 → 开启日历权限。

### Q: AI 助手无法使用？

A: 需要配置 API Key。前往 **设置 → AI 集成**，选择 Provider，填入您的 API Key。DeepSeek、通义千问、Kimi 等各 Provider 的 Key 需从对应官网获取。

### Q: 如何备份数据？

A: 前往 **设置 → 备份 → 导出 JSON**。备份文件在设备下载目录中。建议定期备份并保存到安全位置。

### Q: 删除的模板还能恢复吗？

A: 模板采用"停用"而非物理删除。停用的模板及其历史数据仍在数据库中，但不在列表中显示。如需恢复，请通过备份功能先导出数据。

### Q: 修改了选项名称，历史数据会变吗？

A: 不会。Daily Flow 使用**数据快照**机制——每条记录在保存时已固定选项的 label 和数值，后续修改选项不会影响已有记录。

### Q: 支持同步吗？

A: v1.0 版本不支持云同步。可通过 JSON 备份文件在不同设备间迁移数据。数据库加密和云同步计划在 v1.1/v1.2 实现。

### Q: 离线可用吗？

A: 除 AI 助手外，所有核心功能（任务、日历、自定义记录、统计、备份）完全离线可用。

### Q: 应用在哪些 Android 版本上运行？

A: 最低支持 Android 8.0 (API 26)，已在 Android 9 模拟器和 Android 12+ 上测试。
